Breast cancer detection with machine learning and classification algorithms

#datascience #machinelearning

The project, titled "Breast Cancer Detection using Machine Learning and Classification Algorithms," aims to implement machine learning techniques to analyze data and detect breast cancer. Using a dataset of 569 patients, the study leverages multiple classification algorithms—such as Logistic Regression, K-Nearest Neighbors (KNN), Decision Trees, and Random Forests—along with Self-Organizing Maps to predict breast cancer based on key diagnostic features.

Data Preparation and Cleaning: Data on cell nucleus characteristics were collected through Fine-Needle Aspiration, cleaned of null values, and transformed into a usable format, with critical features like radius, texture, perimeter, and area analyzed to aid in diagnosis.

Self-Organizing Maps: The project employs Self-Organizing Maps, an unsupervised learning method inspired by neurophysics, which organizes patient data in a two-dimensional map to highlight patterns, effectively distinguishing between malignant and benign cases.

Classification Algorithms: Four classification models—Logistic Regression, KNN, Decision Tree, and Random Forest—were implemented to analyze and classify the data. These models underwent training and testing, achieving varied results in terms of accuracy and effectiveness.
